Lexical Analyzer & Parser

C++

This was a project from Automata Theory and Compiler Construction, in which we created the starting foundations of a compiler. Namely, I created the scanner that reads in code, character by character, and stores them in a hashing table. This is later used for the parser that implements a grammar to see if it is a legal sentence in the language. This scanner and parser detects the pseudo-language MINI-P, a miniaturized version of Pascal.

EmuMenu

Lua

A self-directed project that was built with Lua and LOVE 2D 0.8. The goal of this project was to develop an easy-to-use GUI that can be navigated with a gamepad (or a keyboard) that allows me to pick an emulated video game to play from the comfort of my couch without having to use the complicated or time-consuming UI on emulators. It would automatically detect what system the game is for and call the appropriate emulator. When the emulator quits, the menu will pop back up.
